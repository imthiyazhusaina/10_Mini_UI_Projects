# 🌐 10 HTML, CSS & JavaScript Mini Projects

A collection of **10 beginner-friendly web development projects** created to strengthen my understanding of HTML, CSS, and JavaScript fundamentals. Each project focuses on a specific concept such as DOM manipulation, event handling, user interaction, or basic validation.

These projects were built as part of my learning journey and helped me gain practical experience in front-end web development.
